#  SmartBudget - Dashboard Financiero Pro

**SmartBudget** es una aplicación web de gestión de finanzas personales diseñada con un enfoque en la arquitectura limpia y una interfaz moderna de estilo **Glassmorphism**. El proyecto demuestra el manejo de lógica compleja, persistencia de datos y generación de documentos dinámicos.

---

## Características Principales

- **Arquitectura POO**: Uso de clases en JavaScript (`Dato`, `Ingreso`, `Egreso`) para una gestión de datos organizada y escalable.
- **Persistencia con LocalStorage**: Los registros se mantienen guardados en el navegador, simulando una base de datos real.
- **Reportes en PDF**: Integración de la librería `html2pdf.js` para generar reportes financieros descargables con diseño profesional.
- **Interfaz Glassmorphism**: Diseño de vanguardia con efectos de transparencia, desenfoque y total responsividad móvil.
- **Validación de Datos**: Control de entradas para asegurar la integridad de los cálculos y balances financieros.

## Stack Tecnológico

* **Lenguajes:** HTML5, CSS3 (Grid & Flexbox), JavaScript (ES6+).
* **Librerías:** * [html2pdf.js](https://ekoopmans.github.io/html2pdf.js/) (Generación de PDF).
    * [Font Awesome](https://fontawesome.com/) (Iconografía).
    * [Google Fonts](https://fonts.google.com/) (Tipografía Plus Jakarta Sans).
